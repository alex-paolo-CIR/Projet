# Projet ATC

Ce projet est une simulation de contrôle aérien utilisant C++ et SFML.

## Installation de SFML

Le projet est configuré pour chercher SFML dans le dossier suivant :
`C:\Program Files\SFML_3.0.2`

## Compilation et Exécution

1. Ouvrez le dossier "ATC" depuis votre IDE préféré (par exemple, Visual Studio).
2. Assurez-vous que les paramètres de votre projet pointent vers le dossier SFML mentionné ci-dessus.
3. Compilez le projet.

## Structure

*   `assets/` : Contient les ressources du jeu (images, polices). Ce dossier est copié automatiquement à côté de l'exécutable lors de la compilation.
